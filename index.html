PROJECT 2

Nomad Nodes

The Nomad Nodes or Wifi Finder App is a comprehensive mobile and web application designed to facilitate the Wifi activities of the user. This application serves as a robust tool for locating and connecting to available Wifi networks, ensuring users can stay connected wherever they are. The project is structured as a Turborepo, leveraging modern technologies to deliver a seamless cross-platform experience. It includes a React Native app built with Expo for mobile users and a Next.js app utilizing react-native-web for web users. Both applications share a common component library, @repo/ui, which ensures a consistent user interface across platforms.

Key Features of Nomad Nodes or Wifi Finder App

  • Responsive Cross-Platform Design: The app is designed to work seamlessly on both mobile and web platforms, providing a consistent user experience regardless of the device.
  • Relational Database Implementation: A well-structured database supports the app's functionality, enabling efficient data management and retrieval.
  • Mapping Feature: Users can easily navigate from their current location to various Wifi hotspots, thanks to an integrated mapping feature that provides routing capabilities.
  • User Interface Management: The app offers an intuitive and user-friendly interface, making it easy for users to find and connect to Wifi networks.
  • Integration of Routers and Modems: The app supports integration with various routers and modems, enhancing connectivity options for users.
  • Project Management Tools: The development process was streamlined using tools like Trello for task management and Discord/Slack for team communication, ensuring efficient collaboration.
  • Add-On Features: The app includes additional functionalities such as parental controls and ad-blocking capabilities, providing users with enhanced control over their internet experience.

Overall, the Nomad Nodes or Wifi Finder App is a versatile and powerful tool for users seeking reliable Wifi connectivity, backed by a robust technical foundation and a focus on user-centric design.
