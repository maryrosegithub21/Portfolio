PROJECT 1

MCGI FEAST PORTAL

The MCGI Christchurch Portal is a comprehensive web application designed to facilitate the online activities of the MCGI Christchurch community. This portal provides a seamless user experience for members to log in, sign up, and access various features and information about the community. The MCGI Christchurch Portal is a robust and user-friendly application that enhances the online experience for community members. By leveraging modern web technologies and best practices, the portal ensures secure and efficient access to essential features and information.

Key Features

  • User Authentication:
    • Login: Members can log in using their email, password, and church ID. The login process includes validation and error handling to ensure secure access.
    • Sign Up: New members can sign up by providing their details, which are then stored securely in the backend database.
  • Dynamic Routing: The application uses `react-router-dom` to manage navigation between different pages. This includes routes for the home page, login page, and sign-up page.
  • Home Page Components:
    • NavBar: A navigation bar that provides easy access to different sections of the website.
    • HeroBanner: A visually appealing banner that highlights key messages or events.
    • Welcome Section: A section that welcomes users and provides an overview of the community.
    • SlideShow: A slideshow component that showcases images or announcements.
    • Footer: A footer that contains additional links and information.
    • ChatBot: An interactive chatbot that assists users with their queries.
  • Backend Integration:
    • The backend is built using Node.js and MongoDB. The login information is securely stored and managed using Mongoose schemas.
    • The login API endpoint validates user credentials and provides appropriate responses based on the authentication status.
